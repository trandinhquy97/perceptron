# Perceptron
## A library to create a neural cell in a neural network

+ **Init:**
```
var perceptron = new Perceptron(n, lr)
```

n: number of values each input item.
lr: learning rate.

+ **Usage**:
```
perceptron.train(i, e)
```

i: input item (an array).
e: expected value (-1 or 1).

```
perceptron.guess(i)
```

i: input item (an array).
